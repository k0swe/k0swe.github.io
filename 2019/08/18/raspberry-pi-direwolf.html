Raspberry Pi and Dire Wolf

Aug 18, 2019

After getting PiPat to work, I was curious to get something running with more “conventional” hardware. I’d heard a lot about Dire Wolf, so I picked up a Signalink USB sound card/interface device and set about using that with a Raspberry Pi 3 B+ and Pat.

I got it working relatively well. You can see in the photo, the TNC-Pi is still attached to the Raspberry Pi, but it’s not being used for this configuration. I documented my Dire Wolf setup as an appendix in the PiPat documentation.
